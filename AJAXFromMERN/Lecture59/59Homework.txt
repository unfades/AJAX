The countries details app

2 ui one is button load countries,
the 2nd is select in which all countries load.
Then there is a card, which displays country flag,
the region it is in like asia, europe...
the language spoken, english, japanese,
and the currency used in the country like Japanese yen, United States Dollars,
all from the restcountries v2 API
https://restcountries.com/v2/all
